# Cadastro de Clientes em Python

Este projeto é um sistema simples de cadastro de clientes desenvolvido em Python, com execução via terminal. A aplicação permite cadastrar novos clientes e listar todos os clientes cadastrados durante a execução do programa.

## Funcionalidades
- Cadastrar novo cliente
- Listar clientes cadastrados
- Menu interativo no terminal
- Encerramento do sistema

## Tecnologias utilizadas
- Python 3

## Conceitos praticados
- Programação Orientada a Objetos (POO)
- Classes e objetos
- Métodos
- Listas
- Estruturas condicionais
- Estruturas de repetição

## Como executar
1. Certifique-se de ter o Python instalado.
2. Baixe o arquivo do projeto.
3. Execute no terminal:

```bash
python nome_do_arquivo.py
